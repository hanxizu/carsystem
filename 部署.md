部署方案
Vercel 部署
安装 Vercel CLI:

bash
npm i -g vercel
登录并部署:

bash
vercel login
vercel --prod
设置环境变量:

bash
vercel env add WEWORK_CORP_ID
vercel env add WEWORK_CORP_SECRET
vercel env add WEWORK_AGENT_ID
vercel env add TENCENT_SECRET_ID
vercel env add TENCENT_SECRET_KEY
vercel env add SUPABASE_URL
vercel env add SUPABASE_ANON_KEY
Railway 部署
安装 Railway CLI:

bash
npm i -g @railway/cli
登录:

bash
railway login
初始化并部署:

bash
railway init
railway up
添加环境变量:

bash
railway variables set WEWORK_CORP_ID=xxx
railway variables set WEWORK_CORP_SECRET=xxx
# ... 其他环境变量
企业微信配置
登录企业微信管理后台

创建自建应用，获取 CorpId、Secret、AgentId

配置回调URL: https://your-domain/api/webhook

配置接收消息的Token和EncodingAESKey

启动项目
bash
# 开发环境
npm run dev

# 生产环境
npm run build
npm run start
